# Backend FastAPI

## Installation

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

## Lancement

uvicorn main:app --reload --host 0.0.0.0 --port 8000

## Documentation API

Une fois lance, acceder a :
- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc
